# Agente de IA para Configuração de Docker Compose

Este agente de IA foi projetado para auxiliar no desenvolvimento e configuração de arquivos `docker-compose.yml` neste projeto.

## Como Usar:

1.  **Interaja com o Agente:** Faça perguntas ou solicite assistência diretamente sobre o `docker-compose.yml` ou outros aspectos da configuração do Docker.
2.  **Seja Específico:** Forneça o máximo de contexto possível para obter as melhores respostas.
3.  **Revisão:** Sempre revise as sugestões e modificações propostas pelo agente antes de aplicá-las.

Para instruções detalhadas sobre a persona do agente, consulte `agents/persona/DEVELOPER.md`.
